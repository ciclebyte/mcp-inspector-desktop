### Technical Details

**Rust Backend:**
- Tauri 2.x with async command support
- InspectorHandle with automatic process cleanup via Drop trait
- Port auto-selection using portpicker crate (5174-5274, 6277-6377)
- Configuration persistence with atomic writes
- Profile management (create, read, delete)

**Frontend:**
- React 18.3 with TypeScript
- TailwindCSS for styling
- Dark theme by default
- Launcher with file browser support
- Inspector embedding via iframe
